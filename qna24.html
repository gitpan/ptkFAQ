24. What can we expect the future to hold?

Here is a summary of the Tk-b11.02 announcement:

Changes in b11.02

  • Fix for K&R 'cc' (e.g. Sun's /bin/cc) compile.
  • 'Scrolled' scrollbars now only get packed (dynamically) when required.
  • Tweaks for perl5.002_01
  • Tidied up basic_credentials stuff to HTML/Web.pm a bit
  • Brought Pod viewer up to date.
  • Fixed memory leak in Tk::Callback->new, highlighted by recent 'after' changes.
The gamma release is not far off at all.
